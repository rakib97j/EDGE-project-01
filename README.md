# Laravel Blog Project

> **Assignment Project:** This project is prepared for **WDF-1** and **WDF-2 Batch**.

This project implements:
- Common responsive layout (header, footer, main layout) using Tailwind CSS
- Authentication (register, login, logout)
- Post CRUD with draft/published status
- Categories, tags, comments, related posts, and monthly archive
- Dashboard for logged-in users to manage their own posts
- Seeders with sample users, posts, categories, tags, and comments

## Requirements

- PHP `>= 8.4`
- Composer
- Node.js + npm
- MySQL (or another supported DB)

## Installation (Step by Step)

1. Install dependencies:

```bash
composer install
npm install
```

2. Create environment file:

```bash
cp .env.example .env
```

3. Generate app key:

```bash
php artisan key:generate
```

4. Update `.env` database credentials:

```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=blog_project
DB_USERNAME=root
DB_PASSWORD=
```

5. Run migrations with seed data:

```bash
php artisan migrate:fresh --seed
```

## Run the Project

```bash
npm run build
php artisan serve
```

Open: `http://127.0.0.1:8000`

## Main Routes

- Public:
  - `/` posts list
  - `/posts/{slug}` single post
  - `/archive/{year}/{month}` monthly archive
- Authenticated:
  - `/dashboard` post management
  - `/posts/*` create/edit/delete
  - `/comments` add/delete
  - `/categories/*`, `/tags/*`

## Notes

- Public users can view published posts and comments.
- Only logged-in users can create, edit, and delete posts.
- Comment deletion is allowed for:
  - comment owner
  - post owner
  - admin user (`admin@example.com`)
